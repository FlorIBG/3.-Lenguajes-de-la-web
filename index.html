Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML 5

    HTML5 (Hyper Text Markup Language , versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web. HTML.

    Usa la extension .html

  • CSS 3

    Hoja de estilo en casacada o CSS(siglas en inglés de cascading style sheets ) es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.

    Usa la extension css

  • JavaScript

    JavaScript (abreviado comúnmente "JS") es un lenguaje de programacion interpretado utilizado principalmente en el lado del cliente (client-side), implemetado como parte de un navegador web permitiendo menjoras en la interfaz de usuario y páginas web dinámicas.

    Usa la extension .js
